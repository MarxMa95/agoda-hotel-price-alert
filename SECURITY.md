# Security

## Supported use

This repository is intended for local-only usage on a trusted machine.

## Sensitive local data

Do not publish or commit:
- `session_profiles/`
- `data.db`
- `logs/`
- `debug_screens/`

These may contain session material, local paths, captured content, or personal watcher data.

## Reporting

If you discover a security issue in the public code, please report it privately to the maintainer before opening a public issue.
